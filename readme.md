# MailToSlack
## Install

```sh
go install github.com/TKMAX777/MailToSlack
```

## Configuration

### AddEnvironmentVariables

```
SLACK_BOT_TOKEN="xoxb-xxxx"
SLACK_CHANNEL="Cxxxxxxx"
SLACK_ICON_EMOJI=":+1:"
```

### AddMailSettings

```
# /etc/aliases
USERNAME:   "|/path/to/binary"
```
